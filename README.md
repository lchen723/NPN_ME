# NPN_ME
This supplementary material contains programming code and the real dataset for simulation and real data analysis for the manuscript "Estimation of Nonparanormal Graphical Models for Noisy Gene Expression Data" under the consideration in Bioinformatics.

This repository contains
1. GSE158588_counts.csv: A .zip file containing the real dataset.
2. Nonpara_graph_RDA.R: The R script for real data analysis
3. Nonpara_graph_simulation.R: The R script for the scenario "circular arc" in simulation studies 
4. Nonpara_graph_simulation_Hub.R: The R script for the scenario "hub" in simulation studies
